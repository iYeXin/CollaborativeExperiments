# Material Design 3 蓝色风格设计规范

### 色彩系统

- **主色**：`#005EEB`（主要按钮、激活状态）
- **主色容器**：`#E5EFFD`（次要按钮、容器背景）
- **表面色**：`#FBFBFF`（卡片、页面背景）
- **表面文字**：`#333`（主要内容）
- **错误色**：`#BA1A1A`（错误状态）
- **轮廓色**：`#757780`（边框、分隔线）

### 阴影层级

- **轻微阴影**（Elevation 1）：用于应用栏、卡片
- **中等阴影**（Elevation 2）：用于悬停状态
- **较强阴影**（Elevation 3）：用于模态框、对话框

### 形状系统

- **圆角**：小元素 `4px`、按钮 `8px`、卡片 `12px`、大型容器 `16px`、圆形按钮 `28px`

### 动效

- **动画**：持续时间 `0.3s`，缓动函数 `cubic-bezier(0.2, 0, 0, 1)`

### 组件规范

- **应用栏**：表面色背景，主色文字，轻微阴影
- **文本字段**：浮动标签，聚焦时边框变主色
- **按钮**：
  - 填充按钮：主色背景，白色文字，大圆角 (`28px`)，阴影
  - 轮廓按钮：透明背景，主色边框和文字
- **选项卡**：底部激活指示条，主色激活状态
- **表单和列表**：一致圆角，聚焦高亮，悬停背景变化

### 响应式设计

- **断点**：`768px`（中等屏幕）、`480px`（小屏幕）
- **策略**：弹性布局，调整字体大小，隐藏非必要元素

### 交互反馈

- **状态**：悬停轻微背景变化，激活颜色/阴影变化，禁用降低透明度
- **消息**：成功/错误/信息类型，自动隐藏，平滑动画

### 排版与间距

- **字体**：标题 `1.3rem`/`1.25rem`，正文 `1rem`，辅助文字 `0.875rem`/`0.75rem`
- **间距**：以 `8px` 为基础单位，保持一致性

### 图标

- **库**：Font Awesome
- **用法**：图标与文字组合，大小颜色一致，小屏幕隐藏非必要图标
